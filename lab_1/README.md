# Image Processing and NumPy Lab

## 📖 About This Lab
This lab covers basic NumPy operations and image processing using Python.  
We practiced creating arrays, indexing, reshaping matrices, and generating random numbers.

## 🖼 Image Processing Tasks
- Loading images using OpenCV and PIL
- Displaying images using Matplotlib
- Saving images to disk
- Converting images to NumPy arrays
- Viewing image shape and pixel values

## 🛠 Tools Used
- Python
- NumPy
- OpenCV
- Pillow (PIL)
- Matplotlib
- Anaconda (Jupyter / Spyder)

This lab helped in understanding how images are represented as arrays and how to manipulate them using NumPy.
