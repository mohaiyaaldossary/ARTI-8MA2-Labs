# Lab 10: Image Segmentation and Feature Extraction

## Course
ARTI404 – Image Processing

## File Format
Jupyter Notebook (`.ipynb`)

## Description
This lab introduces basic image segmentation and feature extraction techniques using Python and OpenCV. The notebook demonstrates how to detect edges, corners, and separate objects from the background using thresholding methods.

## Topics Covered

### 1. Canny Edge Detection
- Gaussian smoothing
- Gradient computation
- Non-maximum suppression
- Double thresholding
- Edge tracking by hysteresis

### 2. Harris Corner Detection
- Detecting corners in images
- Feature extraction for object tracking and matching

### 3. Binary Thresholding
- Converting grayscale images into binary images using a fixed threshold

### 4. Otsu Thresholding
- Automatic threshold selection
- Separating foreground and background regions

## Libraries Used
- OpenCV (`cv2`)
- NumPy
- Matplotlib
- scikit-image (`skimage`)

## Example Images
- Camera image
- Checkerboard image
- Coins image

## Objectives
- Understand image segmentation techniques
- Apply edge and corner detection
- Compare thresholding methods
- Visualize image processing results

## Output
The notebook displays processed images, detected edges/corners, and segmentation results using plots and visualization tools.
