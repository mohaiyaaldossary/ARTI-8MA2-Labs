# Lab 6 – Frequency Domain Image Filtering

## Description

This project applies **Laplacian** and **Sobel** filters to an image using the **frequency domain** with Fast Fourier Transform (FFT) to detect edges.

## Requirements

* Python
* numpy
* matplotlib
* scikit-image

## Features

* Apply Laplacian edge detection
* Apply Sobel X and Sobel Y filters
* Compute gradient magnitude
* Display original and filtered images

## Learning Goals

* Understand FFT
* Learn frequency domain filtering
* Understand edge detection techniques
