# Image Processing Lab 7

## Overview
This project demonstrates the use of spatial domain filtering techniques in image processing. It includes implementations of the Laplacian filter for edge detection and sharpening, as well as mathematical and statistical filters for noise reduction.

## Files
- `Lab_7_(A)_Laplacian_Filter.ipynb`  
  Applies the Laplacian filter to detect edges and enhance image details.

- `Lab_7_(B)_Maths_&_Statistics_Filters.ipynb`  
  Implements filters such as mean and median for smoothing and noise removal.

## Objectives
- Understand spatial domain filtering  
- Apply Laplacian filter for edge detection  
- Use statistical filters for noise reduction  
- Compare the effects of different filters  

## Tools and Libraries
- Python  
- NumPy  
- OpenCV / PIL  
- Matplotlib  

## How to Run
1. Open the notebooks using Jupyter Notebook or Google Colab  
2. Run the cells sequentially  
3. Observe and analyze the results  

## Notes
- The Laplacian filter highlights edges by detecting intensity changes  
- Mean and median filters are useful for reducing noise and smoothing images
