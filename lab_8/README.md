# Image Processing Lab 8 - Edge Detection

## Overview
This project focuses on edge detection techniques in image processing. It demonstrates how to identify edges in images using different operators and highlights the importance of edges in image analysis.

## File
- `Lab_8_Edge_Detection.ipynb`  
  Implements various edge detection methods such as Sobel, Prewitt, and possibly other gradient-based techniques.

## Objectives
- Understand the concept of edge detection  
- Apply different edge detection operators  
- Analyze how edges represent important features in images  
- Compare the performance of different methods  

## Tools and Libraries
- Python  
- NumPy  
- OpenCV / PIL  
- Matplotlib  

## How to Run
1. Open the notebook using Jupyter Notebook or Google Colab  
2. Run all cells step by step  
3. Observe the detected edges and compare results  

## Notes
- Edge detection is based on changes in pixel intensity  
- Sobel and Prewitt operators use first-order derivatives  
- These methods help in identifying object boundaries in images
